# Product App

React Query seems interesting, so lets try to learn it :=)
## ✔️ Requirements
- NodeJS
- Yarn version 

## 🚀 Installation
1. clone this repository
  ```sh
  git clone https://github.com/jkopi/product-app
  ```
2. change to directory and run command `yarn install`


## 🏃 Run it!
- run command `yarn start`

## 🔧 Built with
- React
- React Query
- Chakra UI
